# 🎀 Birthday sm-esh | 20 ✨

An interactive and magical digital invitation for Smriti & Ananya's 20th Birthday celebration! 🥳

## ✨ Features
- **💌 Magical Envelope**: An animated envelope that opens with a burst of hearts.
- **📜 Sliding Card**: The invitation slides up from the bottom and takes center stage.
- **🎨 Cute Aesthetic**: Pink/Blush theme with floating sparkles and a starry sky.
- **🎶 Bubbly Music**: Built-in background music to set the party mood.
- **✅ RSVP System**: An easy-to-use form that closes the letter and celebrates with more hearts.
- **🕵️‍♀️ Secret Guest List**: A hidden "database" to track RSVPs locally.

## 🚀 How to Use
1. Just open `index.html` in any web browser or open https://smriti-kinra.github.io/sm-esh/ website
2. Tap the envelope to reveal the magic! 🪄

## 🤫 Admin Secrets
To see who has RSVP'd:
- Type the word **"guests"** anywhere on the screen!
- Or click the tiny, faint **"Admin"** link at the very bottom of the page.
